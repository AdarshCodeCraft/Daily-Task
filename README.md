# Daily-Task
<h2>Features:</h2>

1.Task Management: Easily add, edit, or remove tasks from your daily list.

2.Task Status: Mark tasks as completed or pending with a single click.

3.Persistent Data: Saves your tasks in the browser so they stay even after refreshing the page.

4.Interactive UI: A clean and responsive design for desktop and mobile views.

5.Celebratory Animation: A fun, celebratory animation with colorful lights and sparkles when all tasks are completed.

6.Customizable: Simple structure and code that can be easily extended or modified.
